Panora's server-side helper libraries (also known as server-side SDKs) reduce the amount of work required to use Panora's REST APIs, starting with reducing the boilerplate code you have to write. Below are the installation instructions for these libraries in a variety of popular server-side programming languages.

<Tabs>

  <Tab title="TypeScript SDK">
        ```bash Typescript SDK
        npm install panora-sdk
        ```
    </Tab>

    <Tab title="Python SDK">
        ```bash Python SDK
        pip install panorasdk
        ``` 
    </Tab>
 

    <Tab title="Java SDK | Gradle">
        Paste the next line inside the dependencies block of your build.gradle file:

        ```Gradle
        implementation group: "dev.panora", name: "PanoraSDK", version: "0.0.1"
        ```
    </Tab>

    <Tab title="Java SDK | Maven">
        If you use Maven, place the following within the `<dependencies>` tag in your pom.xml file

        ```XML Java SDK Maven
        <dependency>
        <groupId>dev.panora</groupId>
        <artifactId>panora-sdk</artifactId>
        <version>0.0.1</version>
        </dependency>
        ```
    </Tab>

</Tabs>
