## Installation

```bash
pnpm install
```

## Running the app

```bash
# Copy the example env file
cp .env.example .env 

# build
pnpm run build

# development
pnpm run start

# watch mode
pnpm run start:dev

# production mode
pnpm run start:prod
```

## Test

```bash
# unit tests
pnpm run test

# e2e tests
pnpm run test:e2e

# test coverage
pnpm run test:cov
```
